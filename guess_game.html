Number guessing game

Rule of the game:

You are going to guess a number from 1 to 10. If you guessed wrong there will be a hint when you guessed wrong, so you can try again. Try to get a correct answer with the least amount of tries.

    How to play:
  • Input an integer from 0 to 10
  • Press Submit
    Difficulty
  • Easy: 0 to 5
  • Normal: 0 to 10
  • Difficult: 0 to 100
